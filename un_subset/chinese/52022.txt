2003年 出口 货物 、 转让技术 和 提供 技术 援助 ( 管制 ) 法令 ； - 2003年 货物 贸易 ( 管制 ) 法令 ； - 2004年 货物 贸易 ( 禁运 地 ) 法令 。 8. 按照 2003年 针对 刚果 民主 共和国 的 ( 限制性 措施 ) ( 海外 领土 ) 法令 和 2005年 针对 刚果 民主 共和国 的 ( 联合国 制裁 ) ( 海外 领土 ) 法令 ， 还 在 联合王国 海外 领土 执行第 1493 ( 2003 ) 号 决议 及 其后 修正案 。 欧盟 措施 9. 欧盟 通过 共同 立场 和 条例 执行 第1493 ( 2003 ) 号 决议 及 其后 修正案 。 目前 欧洲 联盟 委员会 关于 刚果(金) 限制性 措施 的 第338/2005 号 条例 是 于 2005年 6月 13日 通过 的 ， 随后 经 欧洲 联盟 委员会 第1337/2007 号 条例 和 第666/2008 号 条例 修正 。 10. 直接 适用 于 欧盟 所有 成员国 的 欧盟 条例 禁止 ： -
